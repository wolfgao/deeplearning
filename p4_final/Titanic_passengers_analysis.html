Variable    Definition  Key

- cabin Cabin number    
- embarked  Port of Embarkation C = Cherbourg, Q = Queenstown, S = Southampton

def change_age_category(age):
    if age<=12:
        return 'Child'
    elif (age <= 18 and age > 12):
        return 'Teen'
    elif (age <= 60 and age > 18):
        return 'Adult'
    elif (age>60):
        return 'Senior'
